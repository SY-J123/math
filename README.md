# 수리통계학 정리 노트

교재 내용을 **프린트해서 읽기 위한 HTML 요약 노트**. MathJax로 수식을 렌더링한다.

## 구조

```
assets/
  note.css        공유 스타일 (모든 장 공용)
  _template.html  새 절/새 장 만들 때 복사용 템플릿
노트/
  <장>/<n>-<m>_절제목.html   절별 독립 노트
참고자료/
  <장>/                       교재 스캔 원본 (개인 참고용, 저장소에는 미포함)
```

## 보기

- 각 `.html` 파일을 브라우저로 열면 된다. 같은 절 노트끼리는 하단의 이전/다음 버튼으로 이어 볼 수 있다.
- GitHub는 `.html`을 소스로만 보여주므로, 수식이 렌더된 화면을 보려면 파일을 내려받아 열거나 GitHub Pages를 사용한다.

## 노트 양식

- 박스: **정의 / 정리·성질 / 증명 / 예** + 필요시 부가설명(`.note`)
- 한글 용어 옆 영어 병기, 증명은 단계별로 상세히
- 인쇄(A4) 최적화: 페이저·넓은 수식은 인쇄에서 자동 처리

## 작성 중

- 7장 선형 회귀모형 (7.1 ~ 7.5)
