# 수학 정리 노트

교재 내용을 **프린트해서 읽기 위한 HTML 요약 노트**. 수식은 MathJax로,
기하적으로 표현할 수 있는 부분은 SVG 그림으로 렌더링한다.

## 구조

과목별로 폴더를 나누고, 그 아래 장(章)별로 절 노트를 둔다.

```
assets/
  note.css        공유 스타일 (모든 과목·장 공용)
  _template.html  새 절/새 장 만들 때 복사용 템플릿
notes/
  수리통계학/ch07/7-1_correlation.html ...      (notes/<과목>/<챕터>/n-m_topic.html)
  선형대수/ch04/4-1_transpose-partition.html ...
index.html        과목·장별 목차
references/
  ch04/, ch07/    교재 스캔 원본 (개인 참고용, 저장소에는 미포함 — .gitignore)
```

> 노트는 `notes/<과목>/<챕터>/` 에 두므로 `assets`는 세 단계 위(`../../../assets/note.css`)다.

## 보기

- `index.html`을 열면 전체 목차가 나온다. 같은 장의 절 노트끼리는 하단의 이전/다음 버튼으로 이어 볼 수 있다.
- GitHub는 `.html`을 소스로만 보여주므로, 수식이 렌더된 화면을 보려면 파일을 내려받아 열거나 GitHub Pages를 사용한다.

## 노트 양식

- 박스: **정의 / 정리·성질 / 증명 / 예** + 필요시 부가설명(`.note`)
- 한글 용어 옆 영어 병기, 증명은 단계별로 상세히
- **수식 설명**: 수식을 풀어 설명할 때, 본문 흐름에 이어지는 설명은 별도 스타일·라벨("말로 풀면" 등) 없이
  그냥 본문 텍스트(`<p>`)로 쓴다. 반면 참고·곁가지 정보(개념 보충, 동치 조건 등)는 `.note` 박스로 쓴다.
- 기하적으로 표현 가능한 개념은 `<figure class="fig">` 안에 SVG 그림으로
- 인쇄(A4) 최적화: 페이저·넓은 수식은 인쇄에서 자동 처리
