4.3–4.6대각합·덧셈·스칼라곱·영행렬 trace, addition, scalar, null

4.3행렬의 대각합 trace

정방행렬의 대각원소들의 합을 대각합이라 하고 $tr(A)$ 로 쓴다. $A=(a_{ij}),\ i,j=1,\dots,n$ 에 대해

$$tr(A)=a_{11}+a_{22}+\cdots+a_{nn}=\sum_{i=1}^{n}a_{ii}.$$
예 $$tr\begin{bmatrix}1&7&6\\8&3&9\\4&-2&-8\end{bmatrix}=1+3-8=-4.$$

$A$ 가 정방행렬이 아니면 대각합은 정의되지 않는다. 또한 전치는 대각원소를 그대로 두므로

$$tr(A')=tr(A),$$

스칼라를 $1\times1$ 행렬로 보면 $tr(\text{스칼라})=\text{스칼라}$, 예컨대 $tr(13)=13$ 이다.

4.4행렬의 덧셈 addition

한 컨트리클럽에서 어느 토요일에 회원·손님에게 제공된 점심 수가 $A$, 다음 주 토요일이 $B$ 라 하자.

$$A=\begin{bmatrix}98&24&42\\39&15&22\end{bmatrix},\qquad B=\begin{bmatrix}55&19&44\\43&53&38\end{bmatrix}.$$

두 주에 제공된 각 음식의 총합은 같은 위치 원소끼리 더해 얻는다.

$$A+B=\begin{bmatrix}98{+}55&24{+}19&42{+}44\\39{+}43&15{+}53&22{+}38\end{bmatrix} =\begin{bmatrix}153&43&86\\82&68&60\end{bmatrix}.$$
정의행렬의 합

같은 차수의 $A=(a_{ij})$, $B=(b_{ij})$ 에 대해

$$A+B=(a_{ij}+b_{ij})\qquad(i=1,\dots,r,\ j=1,\dots,c).$$

덧셈은 차수가 같은 행렬끼리만 정의된다. 차 $A-B=(a_{ij}-b_{ij})$ 도 같은 방식이다.

덧셈에 대한 전치와 대각합은 다음을 만족한다.

성질과 증명

(i) 합의 전치 $=$ 전치의 합. $A+B=C=(a_{ij}+b_{ij})$ 라 하면

$$(A+B)'=C'=(c_{ji})=(a_{ji}+b_{ji})=(a_{ji})+(b_{ji})=A'+B'.$$

(ii) 합의 대각합 $=$ 대각합의 합 (단 $A,B$ 는 같은 차수의 정방행렬).

$$tr(A+B)=\sum_{i=1}^{n}(a_{ii}+b_{ii})=\sum_{i=1}^{n}a_{ii}+\sum_{i=1}^{n}b_{ii}=tr(A)+tr(B).$$
∎

4.5스칼라에 의한 곱셈 scalar multiplication

같은 행렬을 더하면 $A+A=(a_{ij})+(a_{ij})=(2a_{ij})=2A$. 양의 정수 $\lambda$ 에 대해 $\lambda A$ 는 $A$ 를 $\lambda$ 번 더한 것이고, 일반적인 스칼라로 확장하면

$$\lambda A=(\lambda a_{ij})\qquad(i=1,\dots,r,\ j=1,\dots,c).$$

즉 모든 원소에 $\lambda$ 를 곱한다. 예를 들면

$$\lambda\begin{bmatrix}2&-7\\3&5\end{bmatrix}=\begin{bmatrix}2\lambda&-7\lambda\\3\lambda&5\lambda\end{bmatrix},\qquad \begin{bmatrix}2&-7\\3&5\end{bmatrix}\!/17=\begin{bmatrix}2/17&-7/17\\3/17&5/17\end{bmatrix}.$$

($\lambda$ 가 분수일 때는 $A/17$ 처럼 나눗셈으로 쓰는 것이 편하다.)

4.6등식과 영행렬 equality & null matrix

두 행렬이 같다는 것은 차수가 같고 대응 원소가 모두 같다는 뜻이다. 즉 모든 $i,j$ 에 대해 $a_{ij}=b_{ij}$. 예를 들어

$$A=\begin{bmatrix}2&6&-4\\3&0&1\end{bmatrix},\quad B=\begin{bmatrix}2&6&-4\\3&0&1\end{bmatrix},\quad C=\begin{bmatrix}2&6&-4\\2&0&1\end{bmatrix}$$

에서 $A=B$ 이지만 $A\neq C$ 이다($(2,1)$ 원소가 다름). 같은 차수의 두 행렬에 대해 $a_{ij}=b_{ij}$ 이면

$$A-B=(a_{ij}-b_{ij})=(0)=0.$$
정의영행렬 null / zero matrix

모든 원소가 $0$ 인 행렬을 영행렬이라 하고 $0$ 으로 쓴다. 행렬대수에서 ‘0’ 의 역할을 한다. 차수마다 영행렬이 따로 있으므로 영행렬은 유일하지 않다. 예를 들어

$$\begin{bmatrix}0&0&0&0\\0&0&0&0\end{bmatrix},\qquad\begin{bmatrix}0&0&0\\0&0&0\\0&0&0\end{bmatrix}.$$
