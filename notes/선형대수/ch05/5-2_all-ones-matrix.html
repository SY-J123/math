5.2모든 원소가 같은 행렬 summing vector & J matrix

모든 원소가 $1$ 인 벡터·행렬은 합·평균 을 행렬곱으로 깔끔하게 표현하게 해 준다. 통계에서 평균·편차제곱합·중심화가 모두 이 도구로 쓰인다.

합벡터 summing vector

정의합벡터 summing vector $1$

모든 원소가 $1$ 인 열벡터를 합벡터라 하고 $1$ (차수를 밝힐 땐 $1_n$)로 쓴다. 벡터 $x$ 와의 내적이 원소들의 합이라 이 이름이 붙었다:

$$1'x=\sum_{i=1}^{n}x_i=x'1.$$
예제 5.5

$1'=[1\ 1\ 1\ 1]$ 은 $1\times4$ 합벡터이고 $x'=[3\ 6\ 8\ -2]$ 에 대해

$$1'x=[1\ 1\ 1\ 1]\begin{bmatrix}3\\6\\8\\-2\end{bmatrix}=3+6+8-2=15=x'1.$$
성질합벡터의 곱
  1. 행렬에 곱하면 열합/행합. $1'_r X_{r\times c}=1'_c$ 꼴로 각 열의 합을 모은 행벡터가 된다(좌우의 $1'$ 은 차수가 다르다 — $1'_r$ 과 $1'_c$). 예: $1'_3\big[\begin{smallmatrix}2&-1\\-5&-3\\4&5\end{smallmatrix}\big]=[\,1\ \ 1\,]=1'_2$.
  2. 자기 내적 = 차수: $\displaystyle 1'_n 1_n=n.\tag{5.13}$

$J$ 행렬 — 모든 원소가 1

정의$J$ 행렬

합벡터의 외적은 모든 원소가 $1$ 인 행렬이며 $J$ (또는 $J_{r\times s}$)로 쓴다:

$$1_r 1'_s=J_{r\times s},\qquad\text{예 } 1_3 1'_2=\begin{bmatrix}1\\1\\1\end{bmatrix}[1\ 1]=\begin{bmatrix}1&1\\1&1\\1&1\end{bmatrix}=J_{3\times2}.\tag{5.14}$$

$\lambda J_{r\times s}$ 는 모든 원소가 $\lambda$ 인 행렬이다. ($J$ 는 $5.1.3$ 절의 외적이므로 rank 1 이다.)

성질$J$ 와 $1$ 의 곱

$J=11'$ 이므로 곱은 가운데 $1'1=(\text{차수})$ 만 스칼라로 남기고 다시 $J$ 나 $1$ 이 된다:

$$J_{r\times s}J_{s\times t}=s\,J_{r\times t},\qquad 1'_r J_{r\times s}=r\,1'_s,\qquad J_{r\times s}1_s=s\,1_r.\tag{5.15}$$

특히 정방인 $J_n=1_n 1'_n$ 은

$$J_n^2=n\,J_n.\tag{5.16}$$
증명

$J=11'$ 로 풀어 결합법칙만 쓰면 된다. 예를 들어 $J_{r\times s}J_{s\times t}=(1_r1'_s)(1_s1'_t)=1_r\underbrace{(1'_s1_s)}_{=\,s}1'_t=s\,1_r1'_t=sJ_{r\times t}$, 같은 방식으로 $J_n^2=1_n(1'_n1_n)1'_n=nJ_n$.

∎

$\bar J$ 와 중심화행렬 centering matrix

정의$\bar J$ 와 중심화행렬 $C$

$\bar J_n=\dfrac1n J_n$ 으로 두면 자기 제곱과 같다(멱등):

$$\bar J_n^2=\frac{1}{n^2}J_n^2=\frac{1}{n^2}(nJ_n)=\frac1n J_n=\bar J_n.\tag{5.17}$$

이때 $C_n=I-\bar J_n=I-\dfrac1n J_n$ 을 통계학의 중심화행렬centering matrix이라 한다.

성질 (5.18)중심화행렬 $C$ $$C=C'=C^2,\qquad C1=0,\qquad CJ=JC=0.$$
증명

대칭·멱등: $\bar J$ 가 대칭이라 $C'=I-\bar J=C$, 그리고 $\bar J^2=\bar J$ 를 쓰면 $C^2=(I-\bar J)^2=I-2\bar J+\bar J^2=I-\bar J=C$. $C1=0$: $J1=11'1=n\,1$ 이므로 $C1=1-\tfrac1n J1=1-1=0$. $CJ=0$: $CJ=(I-\tfrac1nJ)J=J-\tfrac1nJ^2=J-\tfrac1n(nJ)=0$, 대칭으로 $JC=0$.

∎
예제 5.6평균과 편차제곱합 — $1$ 과 $C$ 로

자료 $x'=[x_1\ x_2\ \cdots\ x_n]$ 에 대해 평균은 합벡터로

$$\bar x=\frac{x_1+\cdots+x_n}{n}=\frac1n x'1=\frac1n 1'x.$$

중심화행렬을 오른쪽에서 곱하면 각 자료의 편차가 나온다:

$$x'C=x'-x'\bar J=x'-\frac1n x'11'=x'-\bar x\,1'=\{x_i-\bar x\}.$$

다시 $x$ 를 곱하면 편차제곱합(SS)이 된다:

$$x'Cx=(x'-\bar x\,1')x=x'x-\bar x(1'x)=x'x-\bar x(n\bar x)=x'x-n\bar x^2,$$ $$\sum_{i=1}^{n}(x_i-\bar x)^2=\sum_{i=1}^{n}x_i^2-n\bar x^2=x'x-n\bar x^2=x'Cx.\tag{5.19}$$
식 (5.19) 의 $x'Cx$ 는 제곱합을 나타내는 이차형식quadratic form $x'Ax$ 의 특수한 경우다. $x$ 가 정규분포를 따르면 이런 이차형식은 카이제곱분포를 따르는데, 이를 보이는 데에 멱등성 같은 행렬 개념이 쓰인다. ($C$ 가 대칭·멱등이라 $x'Cx=x'C'Cx=\lVert Cx\rVert^2$ — 편차벡터의 길이의 제곱임이 한눈에 보인다.)
예제 5.7변이확률행렬의 행합 stochastic matrix

변이확률행렬은 각 행의 원소 합이 $1$ 이라, 합벡터를 곱하면 $1$ 이 된다:

$$P=\begin{bmatrix}0.2&0.8\\0.4&0.6\end{bmatrix},\qquad P1=\begin{bmatrix}0.2+0.8\\0.4+0.6\end{bmatrix}=\begin{bmatrix}1\\1\end{bmatrix}=1.$$

나아가 $P^2 1=P(P1)=P1=1$, 일반적으로

$$P^n 1=1$$

이므로 변이확률행렬의 임의의 거듭제곱도 행합이 $1$ 이다 ($n$ 단계 전이확률의 합 $=1$).
